Project Information

Overview

  1. Application Description and Purpose:
    The purpose of this website is to promote a youtube channel and explain how the videos are made
  2. Intented Users:
    The intented users of this website will be the owner of the channel and any viewers who may want to learn more about the creation of the videos
  3. Overview of Content:
    The page will be split between 3 pages. One will be a homepage that features a video and a description of the channel and a small about me section. A second will talk about the recording applications with screenshots and descriptions. A third will talk about the editing and programs used to finish setting up the videos.
